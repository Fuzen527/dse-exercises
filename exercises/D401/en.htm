Given an array of integers $A$, check whether the array is sorted or not.

Input and Output

The first line contains an integer $N$, the size of the array. ($2 \le N \le 100000$)

The $i^\text{th}$ line of the $N$ lines is $A_i$, the $i^\text{th}$ element of array $A$. ($-10^9 \le A_i \le 10^9$).

  • If the elements of array $A$ are all the same, output \texttt{Same}.
  • If the elements of array $A$ are strictly increasing, output \texttt{Strictly increasing}.
  • If the elements of array $A$ are strictly decreasing, output \texttt{Strictly decreasing}.
  • If the elements of array $A$ are increasing (but not the same or strictly increasing), output \texttt{Increasing}.
  • If the elements of array $A$ are decreasing (but not the same or strictly decreasing), output \texttt{Decreasing}.
  • Otherwise, output \texttt{Not sorted}.
