Write a program to evaluate a simple arithmetic expression. The specifications of the expression are as follows:

  • The expression consists 3 operands and 2 operators.
  • The operands are integers between 0 and 100 inclusive.
  • The operators are either \texttt{+}(plus), \texttt{-}(minus) or \texttt{*}(times).
  • The expression does not contain any brackets.

Input and Output

The input consists of one line -- the expression. There will be no spaces between the operands and operators.

Evaluate the expression and output the result.
