Reading input from the standard input (keyboard)

When we run the program, a console window appears. This console window is connected to the standard input and standard output of the program. That is why we can type in our input and also receive the output of the program. When the Enter key is pressed, the input is sent to the program.

Let us take a look at sample 1. The input is 23382338, so we type 23382338 and then press Enter.

The \texttt{readln} function reads input from the standard input. The value obtained will be stored into the variable specified in the parameter. 

readln(a);

The \texttt{scanf} function reads input from the standard input. To use the function, we need to add \texttt{#include } at the beginning of the program.The first parameter of the function is the format string. Here, \texttt{"%d"} specifies that the input would be an integer. Therefore, the value obtained will be stored to the address given after the format string. 

scanf("%d", &a);

The \texttt{Scanner} class will help us read input from the standard input. To use the \texttt{Scanner} class, we need to add \texttt{import java.util.*;} at the beginning of the program. After creating a \texttt{Scanner} object, the input integer value can be obainted by calling its \texttt{nextInt()} method. 

Scanner in = new Scanner(System.in);
a = in.nextInt();

Obtaining the first (most significant) digit of the number

Next, let us extract the first digit of the phone number and store it in another integer variable \texttt{b}. Since the phone number is a 8-digit number, the first digit is equal to the quotient when the phone number is divided by 10,000,000.

In Pascal, we may use the integer division operator \texttt{div}. In C, In Java, we may use the \texttt{/} division operator. When both the operands are integers, it only returns the integer (quotient) part of the result instead of a real number. 

b: integer;
...
b := a div 10000000;
int b = a / 10000000;

After variable \texttt{b} is set appropriately. What are the possible values of \texttt{b}?

Determine the type using \texttt{b}. Specifically, we can use the \texttt{if-else} control structure to do so. The syntax is as follows:

if (condition1) then
  statement1
else if (condition2) then
  statement2
else
  statement3;

if (condition1) {
  statement1;
} else if (condition2) {
  statement2;
} else {
  statement3;
}

When \texttt{condition1} is true, \texttt{statement1} will be executed. Otherwise, the program will skip \texttt{statement1} and continue considering the \texttt{else} cases. Outputting to standard output (console window)

In the previous section, we have finished the conditions. The \texttt{statementx}s inside the \texttt{if} blocks are where we should output the result. Text and values can be sent to the standard output using the \texttt{writeln} function. In Pascal, strings (texts) are enclosed with single quotes. Pass the appropriate string to the \texttt{writeln} function to display the result. Text and values can be sent to the standard output using the \texttt{printf} function. In C, strings (texts) are enclosed with double quotes. Pass the appropriate string to the \texttt{printf} function to display the result. In Java, there are several ways to output text and values to the standard output. One of the ways is to use the \texttt{System.out.println} function. Strings (texts) are enclosed with double quotes.
