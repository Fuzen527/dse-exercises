快速排序法是另一個利用「分治法」的排序算法，與合併排序法不同快速排序法不是把兩個已排序的子陣列合併，而是利用一個中心點 (pivot)，把其餘的元素分成兩部分，小於或等於中心點的和大於中心點的，然後分別遞歸排序兩部分。

在這練習中，我們會實現 Lomuto partition scheme。以下是分割陣列 $a[l..r]$ 的步驟:

  1. 選取最尾的元素 $a[r]$ 為中心點 $p$。另一變量 $i$ (初始值為 $l$) 用來記錄當有 $a[j] \le p$ 時，應將該值放置在的位置。
  2. 迴圈 $j = l, l + 1, \dots, r - 1$: 如果值 $a[j] \le p$，則應將它放置於左邊，方法是將 $a[j]$ 與 $a[i]$ 交換，然後將 $i$ 增加 $1$。如果 $a[i] \gt p$，則無須做任何事，因為比中間點小的值，可在之後與它交換。例如，對於陣列 $a[1..4] = (5, 3, 1, 4)$、$p = a[4] = 4$，在第一步我們無須把 $a[1] = 5$ 移動。在第二步，$a[2] = 3$ 與 $a[i] = 5$ 交換，陣列變成 $(3, 5, 1, 4)$ 且 $i$ 改為 $2$。在第三步，$a[3] = 1$ 與 $a[i] = 1$ 交換，陣列變成 $(3, 1, 5, 4)$ 且 $i$ 改為 $3$。
  3. 由此可見，經過以上之後陣列分成了三部分: $\le p$ 的元素，接著 $\gt p$ 的元素，及元素 $p$。現在我們須把 $a[r] = p$ 放置到正確的位置上，最簡單方法時把 $a[r]$ 與 $a[i]$ 交換。對於上述例子，陣列最終為 $(3, 1, 4, 5)$。

編寫一個程式按以上方法把一個陣列分割，並輸出結果。

輸入

第一行包含兩個整數 $N$ 和 $A$。 $N$ 是陣列的數字數目 ($1 \le N \le 2000$)。

下一行是該陣列，$N$ 數字以空格分隔，全部整數都在 $[-2147483648, 2147483647]$ 內 (32 位元整數範圍)。

輸出

在一行輸出完成分割的陣列，該 $N$ 個數應用空格分隔。
