In mathematics, the Fibonacci sequence is defined as $F_n = F_{n-1} + F_{n-2}$ with $F_0 = 0$ and $F_1 = 1$. The first few Fibonacci numbers in the sequence are 0, 1, 1, 2, 3, 5, 8, 13, ....

Write a program to find F_x.

Input and Output

The input consists of an integer $x$. $(0 \le x \le 40)$

Output $F_x$
