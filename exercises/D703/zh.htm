鏈表是一種能有效地順序存取、插入和刪除資料的數據結構。當我們在鏈表移動時，須利用一個指示標來記錄現在的位置，它支援以下的操作：

  • \texttt{HEAD}: 重置指示標至鏈表的頭端
  • \texttt{NEXT}: 移動指示標至下一元素
  • \texttt{QUERY}: 傳回指示標指著的元素的值

在這練習中，我們會學習陣列如何能表示一個鏈表。編寫一個程式，在鏈表上移動存取。

輸入

第一行包含兩個整數 $N$ 和 $H$， $N$ 是儲存該鏈表的陣列的大小 ($1 \le N \le 10000$)， $H$ 是鏈表第一個元素所在的位置 ($0 \le H \le N$)，如果 $H = 0$，代表鏈表是空白的。

下一行包括 $N$ 個整數，其中第 $i$ 個數是陣列位置 $i$ 的元素儲存的值，這些整數都在 $1$ 和 $10^9$ 之間。

下一行包括 $N$ 個整數，其中第 $i$ 個數是陣列位置 $i$ 的元素的指示標，即緊接著的元素所在的陣列位置。對於鏈表的最後一個元素，該數字會是 0。 可能存在一些位置是不能到達的，它們可能是被刪除或未曾使用，在這些位置可能儲存著任何值或指示標。

輸出

由開端開始，逐一在獨立的行上輸出鏈表內的值。

最後在另一行輸出 \texttt{End}。
