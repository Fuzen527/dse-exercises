Write a program to determine whether $N$ is a square number or a triangular number.

Input and Output

The input consists of the integer $N$. $2 \le N \le 10^18$.

If $N$ is a square number, output \texttt{Square}.

If $N$ is a triangular number, output \texttt{Triangular}.

If $N$ is neither a square number nor a triangular number, output \text{Neither}.
