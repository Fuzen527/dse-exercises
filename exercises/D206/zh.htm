奇偶歸一猜想，也被稱為 $3N+1$ 猜想指出：給定一個開始的正整數$N$，我們反覆進行以下操作之後最終得到$N=1$：
$N \leftarrow \begin{cases} N/2 &\text{if } N \equiv 0 \pmod{2}\\ 3N+1 & \text{if } N\equiv 1 \pmod{2} .\end{cases}$

例如，如果我們從 $N=13$ 開始，序列應該是：13、40、20、10、5、16、8、4、2、1

寫出一個程式，輸出該序列。

輸入與輸出

輸入為一個整數 $N$。($1 \le N \le 100000$)

輸出該起始於$N$而完結於 1 的序列，每個數應該輸出在獨立的行上。
