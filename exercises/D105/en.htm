Write a program to compare two dates: $y_1 \dash m_1 \dash d_1$ and $y_2 \dash m_2 \dash d_2$.

Input and Output

The input consists of two lines. The first line contains $y_1$, $m_1$ and $d_1$ separated by spaces. The second line contains $y_1$, $m_1$ and $d_1$ separated by spaces.

$1000 \le y_1, y_2 \le 9999$, and the dates are guaranteed to be valid.

If $y_1 \dash m_1 \dash d_1$ is earlier, output \texttt{Before}.

If $y_2 \dash m_2 \dash d_2$ is earlier, output \texttt{After}.

If the two dates are the same, output \text{Same}.
