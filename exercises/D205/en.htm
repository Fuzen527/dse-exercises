A prime number is an integer greater than 1 that is divisible by 1 and itself only. An integer that is greater than one and is not prime numbers is called a composite number.

Write a program to determine whether a number $N$ is prime or composite.

Input and Output

The input consists of an integer $N$. $(2 \le N \le 10^9)$

If $N$ is prime, output \texttt{Prime}. If $N$ is composite, output \texttt{Composite}.
