Bubble Sort sorts an array by repeatly comparing adjacent elements, and swap them if required. Implement the Bubble Sort sorting algorithm. Sort the given array in the specified order.

Input

The first line contains two integers, $N$ and $A$. $N$ is the number of integers in the array ($1 \le N \le 5000$). $A$ is the sorting order required (0 = Ascending, 1 = Descending).

The next line contains an array of $N$ integers separated by spaces. The integers are in $[-2147483648, 2147483647]$ (the range of 32-bit integer). The integers may not be the same.

Output

Output one integer on the first line: the minimum number of swaps required.

On the second line should contain the sorted array. Output $N$ numbers separated by spaces.
