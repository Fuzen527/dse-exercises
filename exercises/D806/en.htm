Implement the Quick Sort sorting algorithm. Sort the given array in the specified order.

Input

The first line contains two integers, $N$ and $A$. $N$ is the number of integers in the array ($1 \le N \le 100,000$). $A$ is the sorting order required (0 = Ascending, 1 = Descending).

The next line contains $N$ integers separated by spaces. The integers are in $[-2147483648, 2147483647]$.

Output

Output the sorted array in one line. The $N$ integers should be separated by spaces and sorted in ascending order.
