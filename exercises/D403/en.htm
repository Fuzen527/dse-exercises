You are playing Scrabble. Write a program to help you check which words can be formed using your tiles.

Input

The first line contains a string $S$ of uppercase letters. Each character in the string represent a tile you have. The length of $S$ is between 1 and 10 inclusive.

The next line contains an integer $N$, the number of words to be checked. ($1 \le N \le 100$).

Each of the next $N$ lines contain a word to be checked. Each word contains 1 to 10 uppercase letters.

Output

For each word, check whether the word can be formed with the tiles you have, and output \texttt{Yes} or \texttt{No}.
